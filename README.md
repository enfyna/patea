# Building

1. Clone the repo.

        git clone https://github.com/enfyna/patea
        cd patea

2. Install dependencies.

        sudo nala install make pkg-config sqlite3 libsqlite3-dev libvte-2.91-dev

3. Build the project.

        make

4. Run the project.

         ./build/patea
